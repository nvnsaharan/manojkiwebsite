Applied Machine Learning Projects:
Regression,
Image Recognition, and
Text Sentiment Prediction

Worked on three distinct projects focussing on three major domains in applied machine learning as described:

1. Comparative prediction of house price using Linear Regression, LassoCV, ElasticNet, RidgeCV, and XGBoost. Used the 'House Sales in King Country, USA dataset' to train the models. XGBoost was found to be the most effective with the least error.

2. CNNs for sign language recognition from hand gestures. Used the 'Sign Language MNIST Dataset' for training a neural network to recognize the English alphabet corresponding to various hand-made gestures in sign language. The base model achieved an accuracy of 99.60%.

3. Comparative text emotion detection using Naive Bayes, Linear SVM, Logistic Regression, and Random Forest models. Applied various NLP techniques on a labeled Twitter tweets dataset and trained models for recognizing if a tweet represents a happy or a sad emotion. The linear SVM model was found to achieve the highest accuracy.
